About the project

Built on a modern Retrieval-Augmented Generation pipeline

PaperLens AI turns static PDFs into a conversational knowledge source. When a document is uploaded, it's split into overlapping chunks, embedded with a sentence-transformer model, and stored inside a vector database. Every question you ask is embedded the same way, matched against the most relevant chunks, and passed to a large language model that answers strictly from that retrieved context.

The result: answers that are fast, explainable, and traceable back to the exact page they came from.

  • No hallucinated citations — every source is real
  • Works with multi-page, text-based PDFs
  • Fully client-rendered, framework-free interface
